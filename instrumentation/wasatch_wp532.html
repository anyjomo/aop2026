Notes

- All optical inputs feed `helpers.compute_fluence_table` which converts
  the nominal mW power axis into calibrated peak irradiance [W/cm^2] and
  cumulative fluence [J/cm^2] via the Gaussian beam waist.
- Adjust `numerical_aperture`, `M_squared`, and `spot_diameter_um` to the
  actual objective in use; the rest of the pipeline does not change.
- Per-acquisition values (laser power, integration time, scan averaging)
  are auto-extracted from ENLIGHTEN raw CSV metadata when present; the
  `acquisition_defaults` block is only used as a fallback.
